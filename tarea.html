1.¿Qué es un Protocolos Http?

Es el nombre de un protocolo el cual nos permite realizar una petición de datos y recursos, como pueden ser documentos HTML.

2.¿Qué es un Query String?

Es un término informático que se utiliza para una interacción con una base de datos. Es la parte de una URL que contiene datos que deberan pasarse a aplicaciones web como los programas CGI.

3.¿Qué es un Verbo HTTP?

Los verbos HTTP le indican al servidor qué hacer con los datos identificados por la URL.

4.¿Qué es un DNS?

Es un conjunto de protocolos y servicios que permite a los usuarios utilizar nombres en vez de tener que recordar direcciones IP numéricas.

5.¿Qué es un CDN?

(red de distribución de contenido) Es una red de servidores que distribuye contenido de un servidor de "origen" a todo el mundo, almacenando ese contenido en caché en un lugar más cercano al de cada usuario final que accede a Internet con un dispositivo habilitado para la Web.
